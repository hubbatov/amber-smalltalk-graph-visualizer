Data structure:

- User
	- firstName				(String)
	- lastName				(String)
	- password				(String)
	- gender				(Boolean)
	- friendList				(OrderedCollection <User>)
	- picture				(Path to image)
	- login 				(String)
	unique (login )

- Client representation
	- node					(GraphNode) - name = (login)
	- edge					(GraphEdge) - scr : dest = (login)
	- addFriend button			(AddEdge)
	- removeFriend button			(RemoveNode)
	- currentUser				(String) - (login)
	- authorization				(Login)
	- exit					(Logout)
	- registration				(Register)


- Server
	- Starting server
	- Stopping server
	- Emit event (Add friend, Remove friend)
	- Commands:
		- Add new user (registration)
		- Add friend
		- Remove friend
		- Authorization
		- Logout
